# CVWO-Frontend 
**ReadIT Web Application [2026]**

## About 
A web application for blogging

## Instructions to set up the frontend server

1. **Clone the repository**  
    ```bash
    git clone <repository-url>
    ```   
2. **Open terminal and navigate inside the project folder**
   ```bash
    cd cvwo-frontend-react
    ```

3. **Install the required Node modules**  
    ```bash
    npm install
    ```

4. **Run the development server**  
    ```bash
    npm run dev
    ```
5. **Open your browser and visit: (ensure backend setup is complete)**
   ```bash
    http://localhost:5173
    ```
